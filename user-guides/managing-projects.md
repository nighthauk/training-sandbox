# Managing Projects

Learn how to create, organize, and manage your projects effectively.

## Creating a New Project

1. Click the **New Project** button in your dashboard
2. Enter project details:
   - Project name (required)
   - Description (optional)
   - Category
   - Privacy settings (public/private)
3. Click **Create Project**

## Project Settings

### General Settings

Configure basic project information:

- **Name**: Update project name
- **Description**: Add or edit project description
- **Tags**: Add searchable tags
- **Status**: Active, Archived, or On Hold

### Collaboration Settings

Manage who can access your project:

- **Owner**: Full control (you)
- **Admin**: Can modify settings and add members
- **Member**: Can view and edit content
- **Viewer**: Read-only access

### Adding Team Members

1. Go to Project Settings > Team
2. Click **Add Member**
3. Enter email address
4. Select permission level
5. Click **Send Invitation**

## Organizing Projects

Use folders and tags to keep projects organized:

- Create folders for related projects
- Use consistent naming conventions
- Apply relevant tags for easy searching
- Archive completed projects

## Deleting Projects

**Warning:** Deleting a project is permanent and cannot be undone.

To delete a project:

1. Go to Project Settings
2. Scroll to **Danger Zone**
3. Click **Delete Project**
4. Confirm by typing the project name
5. Click **I understand, delete this project**

## Best Practices

- Use descriptive project names
- Keep project descriptions up to date
- Regularly review and archive old projects
- Set appropriate permissions for team members
- Use tags consistently across projects
