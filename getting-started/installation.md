# Installation Guide

This guide covers installation steps for different platforms.

## System Requirements

**Minimum Requirements:**
- Operating System: Windows 10, macOS 10.15+, or Linux
- Memory: 4GB RAM
- Storage: 500MB free space
- Internet connection

**Recommended:**
- 8GB RAM or more
- SSD storage
- Broadband internet connection

## Windows Installation

1. Download the installer from our website
2. Run the `.exe` file
3. Follow the installation wizard
4. Launch the application from Start Menu

## macOS Installation

1. Download the `.dmg` file
2. Open the disk image
3. Drag the application to Applications folder
4. Launch from Applications or Spotlight

## Linux Installation

### Using Package Manager (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install our-product
```

### Manual Installation

```bash
wget https://downloads.example.com/product-latest.tar.gz
tar -xzf product-latest.tar.gz
cd product
sudo ./install.sh
```

## Verification

After installation, verify by running:

```bash
product --version
```

You should see the version number displayed.
