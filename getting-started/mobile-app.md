# Mobile App Guide

Learn how to use our mobile application on iOS and Android.

## Installation

### iOS (iPhone/iPad)

1. Open the App Store
2. Search for "Product Name"
3. Tap **Get** then **Install**
4. Open the app after installation

**Requirements:**
- iOS 14.0 or later
- 100MB free storage
- Active internet connection

### Android

1. Open Google Play Store
2. Search for "Product Name"
3. Tap **Install**
4. Open after installation completes

**Requirements:**
- Android 8.0 or later
- 100MB free storage
- Active internet connection

## First Launch

When you first open the app:

1. Accept permissions (notifications, camera if needed)
2. Sign in with your account
3. Complete the quick tutorial
4. Sync your data from the cloud

## Mobile Features

### Offline Mode

Work without internet connection:
- View previously synced projects
- Make changes offline
- Auto-sync when connection restored

### Push Notifications

Stay updated with:
- Project updates
- Team mentions
- Task reminders
- System announcements

Configure in Settings > Notifications

### Mobile-Specific Features

- **Quick Capture**: Tap camera icon to add photos/notes instantly
- **Voice Input**: Use voice-to-text for descriptions
- **Touch ID/Face ID**: Secure login with biometrics
- **Widgets**: Add home screen widgets for quick access

## Tips for Mobile Use

1. **Enable auto-sync** for real-time updates
2. **Use widgets** for quick access to frequent tasks
3. **Enable offline mode** for travel
4. **Use gestures** - swipe to delete, long-press for options
5. **Optimize battery** - reduce sync frequency if needed

## Troubleshooting

### App Crashes on Launch

1. Force quit and restart
2. Check for app updates
3. Restart your device
4. Reinstall if issues persist

### Sync Issues

1. Check internet connection
2. Ensure you're logged in
3. Try manual sync (pull down to refresh)
4. Sign out and back in

## Differences from Desktop

Mobile app has some limitations:
- No bulk editing
- Limited export options
- Simplified settings menu
- Some advanced features require desktop

For full functionality, use the desktop application.
